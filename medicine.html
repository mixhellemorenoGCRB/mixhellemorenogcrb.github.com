What is an information system?

An information system is a purpose built system designed to collect, store, process, distribute and use information

In a hospital an information system would be used to:

  • Appointment scheduling
  • Manage medication dispensing
  • Facilitate communication between doctors, nurses and other healthcare provider
  • Public health reporting
  • Information systems are used by healthcare professionals for making informed decisions about patient care
